<h2 id="uso">:keyboard: &nbsp; Instalação e Uso</h2>

+ Instale as dependências:
```
$ npm install
```

+ Inicie a aplicação:
```
$ npm run dev
```

A aplicação estará disponível em [localhost:3000](localhost:3000/) e a API estará disponível em [localhost:8080](ocalhost:8080/).
